# How to run

### Backend API service
Backend is built with Sails.js
```sh
$ npm install
$ node app.js
```
demo APIs will be accessible at

 http://localhost:1337/api

### Angular 6 Application

```sh
$ npm install
$ ng serve
```
 now the project will be accessible at

 http://localhost:4200/
